# Riff Showcase Release Plan

This file locks the release-ready arrangement and visual sequence for the public demo video.

## Demo Target

- Runtime: 2:30-3:00
- Format: 1920x1080, 30 fps, stereo audio
- Story: "Idea -> AI arrangement -> MIDI edits -> mix polish -> final bounce"

## Arrangement Lock

- Global:
  - BPM: 120
  - Time signature: 4/4
  - Key: A minor
- Section map:
  - Bars 1-8: Intro (drums + pad + bass fade-in)
  - Bars 9-24: Groove A (full rhythm section + motif lead)
  - Bars 25-32: Breakdown (drums filtered, bass sparse, no lead)
  - Bars 33-48: Groove B (chord swap variation + wider drums)
  - Bars 49-56: Build (automation rise, fill, delay throws)
  - Bars 57-72: Final chorus (all layers, strongest hook)
  - Bars 73-80: Outro (drums drop, pad + bass tail)
- Track expectations:
  - Kick/snare/hat/percussion on separate drum tracks
  - Bass on dedicated audio or MIDI track
  - Harmony pad/chords on one track
  - Lead/hook on one track
  - One FX riser/downlifter track
  - Optional vocal chop or texture layer at <= -18 dB RMS
- Mix readiness checkpoints:
  - No master clipping in playback
  - Peak headroom >= 3 dB before limiter
  - LUFS target for demo export: around -14 LUFS integrated
  - No muted placeholder tracks
